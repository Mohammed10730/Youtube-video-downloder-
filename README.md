# 🎥 YouTube Video Downloader

A simple and fast tool to download YouTube videos in different qualities using Python.

---

## 🚀 Features

* Download YouTube videos
* Choose video quality (720p / 1080p / audio only)
* Extract audio as MP3
* Simple interface (CLI or Web with Flask)
* Fast and reliable

---

## 🛠️ Requirements

* Python 3.8+
* pip

---

## ▶️ Usage

Run the program:

```bash
python main.py
```

Then:

* Paste the YouTube video URL
* Choose the desired quality
* Download will start automatically

---

## ⚙️ Libraries Used

* yt-dlp
* Flask (optional for web interface)
* requests
